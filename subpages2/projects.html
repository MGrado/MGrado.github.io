Class Projects:

Python

In the fall semester of my freshman year, I learned the Python coding language. For my final project, I analyzed a dataset from the years 2020 to 2022 to see how children's home lives in Connecticut affect their attendance rate at school. Through linear regression models and other statistical testing, I learned that although students are mentally drained from uncontrollable outside of school, their overall attendance, with regards to this dataset, was unaffected.

Java

In the spring semester of my freshman year, I learned how to code in Java. For my final project, I developed a 211-hotline model that was interactive through Java and JavaFX. This fluctuating box model allowed for users to view the amount of hotline calls for specific years and situations. This was accomplished through developing buttons, drop-down menus, and user input boxes.

PostgreSQL

In the fall semester of my sophomore year, I learned to code in PostgreSQL. For my final project, I developed a mock dataset of a grocery store. Through this dataset, I developed queries and trigger functions that would mimic how a user would utilize this online grocery store.
